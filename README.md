## 💻 Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/guisalva/rs-fullstack-zingen.git
```

Entre no diretório do projeto

```bash
  cd rs-fullstack-zingen
```


Utilizar o **Live Server** ou abrir o `index.html` com o navegador de sua preferência
